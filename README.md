# 아이젠하워 투두 리스트 PWA

아이젠하워 매트릭스(Q1~Q4) 기반으로 일정을 분류하고, 반복/날짜/시간/알림/세부 체크리스트/완료 숨김 기능을 제공하는 개인용 PWA입니다.

## 실행

```bash
npm install
npm start
```

브라우저에서 `http://localhost:3000` 접속.

## 주요 기능

- 일정 CRUD (제목/설명/분류)
- 4분면(중요/긴급) 표시
- 날짜/시간 설정
- 반복: 없음/매일/매주/매월(요일/일자 검증)
- 체크박스 완료 처리 + 취소선
- 완료 일정 숨김 토글(설정 저장)
- 세부 체크리스트 추가/삭제/완료
- 기본 알림(Notification API, 날짜+시간 필요)
- PWA(Manifest + Service Worker 캐시)

## 저장소

- Task/Settings는 `localStorage`에 저장됩니다.
- 오프라인 기본 동작은 캐시된 정적 파일 + 로컬 저장 데이터로 동작합니다.
