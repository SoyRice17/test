# 아이젠하워 투두 리스트 PWA

아이젠하워 매트릭스(Q1~Q4) 기반 일정을 모바일 중심으로 관리하는 개인용 PWA입니다.

## 실행

```bash
npm install
npm start
```

브라우저에서 `http://localhost:3000` 접속.

## 주요 기능

- 모바일 앱 느낌의 UI (글래스 헤더, 카드형 분면, FAB 추가 버튼)
- 빠른 필터(전체/진행중/완료/오늘) + 실시간 검색
- 일정 CRUD (제목/설명/분류)
- 4분면(중요/긴급) 커스텀 라벨
- 날짜/시간, 반복(매일/매주/매월)
- 체크리스트 + 완료 처리
- 완료 숨김 토글, 로컬 저장
- 기본 알림(Notification API, 날짜+시간 필요)
- PWA(Manifest + Service Worker 캐시)

## 저장소

- Task/Settings는 `localStorage`에 저장됩니다.
- 오프라인 기본 동작은 캐시된 정적 파일 + 로컬 저장 데이터로 동작합니다.
